# 課題１２

## Webアプリケーションの動かしかた

1. 準備
以下のコマンドを実行して必要なライブラリをインストールする
```
 % npm install
```
* これは最初に一回やればOK

2. 実行
以下のコマンドを実行する
```
% node index.js
```
http://localhost:3000　をブラウザでひらけばWebアプリを見ることができます。


## もしエラーになったら
 better-sqlite3 のネイティブバインディングがビルドされていない
というエラーが出ることがあるようです。その場合は以下のコマンドを
実行してみてください

```
% npm rebuild better-sqlite3
```
